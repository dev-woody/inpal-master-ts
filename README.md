## front server 실행하기

하기 코드를 터미널에 입력하여 실행시켜주세요.

```
yarn install
yarn start
```

### 이 후에 Open [http://172.30.1.55:3000](http://172.30.1.55:3000) 를 통해서 테스트를 실행합니다.
